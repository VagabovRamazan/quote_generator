# Random Quote Generator

**Автор:** Вагабов Рамазан Рабадан-гаджиевич 

## Описание

Random Quote Generator — это десктопное приложение на Python с графическим интерфейсом для генерации и управления цитатами. Приложение позволяет просматривать цитаты разных авторов и тем, сохранять историю просмотренных цитат с возможностью фильтрации, а также добавлять новые цитаты в базу данных.

## Особенности

- 🎲 Генерация случайных цитат
- 📜 История всех сгенерированных цитат
- 🔍 Умная фильтрация — при выборе фильтра история автоматически показывает только подходящие цитаты
- ➕ Добавление новых цитат с валидацией ввода
- 💾 Автоматическое сохранение всех данных в JSON файл
- 🖥️ Удобный интерфейс с прокруткой
- 🔄 Автоматическое применение фильтров при выборе

## Как это работает

### Фильтрация
При выборе автора или темы из выпадающего списка:
- История автоматически фильтруется, показывая только подходящие цитаты
- Количество отфильтрованных цитат отображается в интерфейсе
- Кнопка "Сгенерировать цитату" генерирует случайную цитату из отфильтрованной базы
- При сбросе фильтров показываются все цитаты из истории

### История
- Хранит все когда-либо сгенерированные цитаты
- При фильтрации показывает только релевантные записи
- Сохраняется между сеансами работы в JSON файле
- Показывает последние 20 цитат (или меньше при фильтрации)

## Требования

- Python 3.6 или выше
- Только стандартные библиотеки Python:
  - `tkinter` — для графического интерфейса
  - `json` — для сохранения данных
  - `random` — для генерации случайных цитат
  - `os` — для работы с файлами
